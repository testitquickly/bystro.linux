====== Как узнать свой IP ======

[ @console @ip ]

===== Internal IP =====

''hostname -I | awk '{print $1}''' 

===== External IP =====

Простейший способ:

''curl ifconfig.me''

Веселейший способ:

''curl wtfismyip.com/json''

См. [[Manjaro:HowTo:Узнать свой внутренний IP]] (там отдельный подход)
