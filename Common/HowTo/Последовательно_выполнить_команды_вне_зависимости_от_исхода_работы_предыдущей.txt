====== Последовательно выполнить команды вне зависимости от исхода работы предыдущей ======

< @console >

Выполнить сколько угодно команд последовательно можно путем разделение команд символом ; (точка с запятой). Пример:

''./configure ; make ; make install''

Выполняются все команды последовательно БЕЗ зависимости от исхода предыдущих. То есть, если после “./configure” будет выведено сообщение об ошибке, то интерпретатор все равно попробует выполнить все следующие команды. Если нужна зависимость — [[Последовательно выполнить команды при условии успешного завершения предыдущих]]
