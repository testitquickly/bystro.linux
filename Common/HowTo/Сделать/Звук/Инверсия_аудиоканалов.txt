====== Инверсия аудиоканалов ======

[ @audio @pulse ]

Иногда надо поменять местами аудиоканалы.

===== Включить =====

Можно временно включить инверсию аудиоканалов через загрузку нового модуля для PulseAudio (с условным названием ”swapped”):

''pactl load-module module-remap-sink sink_name=swapped channels=2 channel_map=right,left''

Ожидаемый ответ: ID запущенного модуля, например ''24''.

Сделать стерео-чек: https://www.youtube.com/watch?v=6TWJaFD6R2s

Временная настройка распространяется на ВСЕ приложения и будет работать до следующего перезапуска PulseAudio (или всей системы).

===== Отключить =====

Надо выгрузить временно добавленный модуль, ориентируясь по его номеру. 

Если забыл ID:

''pactl list short modules | grep module-remap-sink''

Выгрузить:

''pactl unload-module 24''

Временный модуль с инверсией каналов будет полностью убран и стереовывод вернётся к исходной конфигурации. Приложение, которое уже захватило поток вывода, скорее всего, надо перезагрузить.
