====== Монитор ======

Если в мониторе есть режим DDC/CI, то можно управлять настройками монитора через командную строку.

	Управление яркостью, контрастностью и другими параметрами без использования кнопок на мониторе.
	Выбор входного источника сигнала.
	Чтение информации о модели, серийном номере и других характеристиках монитора.
	Упрощённая настройка нескольких мониторов через программное обеспечение.

Управления настройками монитора через командную строку — ddcutil (Linux): Утилита для управления настройками монитора через командную строку.
https://www.ddcutil.com/#introduction

Для Windows есть «Monitorian» и «ClickMonitorDDC».
