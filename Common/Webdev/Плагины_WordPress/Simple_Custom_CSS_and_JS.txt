====== Simple Custom CSS and JS ======

[ @wordpress @js ]

На сервер с WP можно закинуть js-файл по ftp, но его выполнение будет заблокировано. JS-скрипты надо «регистрировать» внутри WP. Это делается через плагины вроде «Simple Custom CSS and JS».

Если в админке русский язык, плагин будет называться «Произвольные CSS и JS».

Если на сайте есть плагины, управляющие кэшем (вроде «W3 Total Cache» или «WP Fastest Cache»), вероятно, сперва надо будет сбросить кэш.

Сперва надо убедиться в том, что на сервере есть каталог „wp-content/uploads/custom-css-js” и он открыт на запись

1. В админке WP пройти в Плагины > «Произвольные CSS и JS» > Добавить произвольный JS
2. Обязательно добавить заголовок нового скрипта (произвольный)
3. Вставить текст скрипта
4. Выбрать «Тип связывания» — «Внешний файл»
5. Отредактировать «Permalink» — переименовать во что-то внятное
6. [Опубликовать]

Скопировать «Permalink», в html-файл, в котором должен работать новый js-скрипт, вставить инструкцию по подключению js-файла через «Permalink». 

Обычное место — между [/style> и [/head]:

''[/style]''
''[script type="text/javascript" src="Мой-Permalink-Полностью"> [/script]''
''[/head]''
