====== WP Mail SMTP ======

[ @wp @wp_plugins @mail ]

Обслуживает [[Noptin]].

Настройка авторизации на стороне моего провайдера

* Принудительно использовать адрес отправителя: вкл.
* Адрес возврата (return-path): вкл
* Почтовая программа: прочие SMTP
	* SMTP сервер: см. настройки хостера
	* Шифрование: SSL
	* SMTP порт: см. настройки хостера
	* Аuto TLS: вкл
	* Аутентификация: вкл
		* Имя пользователя SMTP / Пароль SMTP
[Сохранить настройки]
