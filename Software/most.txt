====== most ======

[ @most ]

Утилита (пейджер) для удобного просмотра текстового файла или вывода чего-то в консоль. Она пришла из времен, когда перед глазами был один экран (черный, консоль же), а переключение между окнами не работало, потому что в компьютерах не было концепции оконного интерфейса. 

Когда в файле (или в выводе команды) много строк, на экране остаётся вывод только последних строк, все остальные улетают куда-то на несколько экранов вверх. Тонелирование вывода в консоль через most (''most file.txt'') вызывает экран чтения документации man — с переносами, отступами и вероятным оформлением bold, italic и прочего, если в оригинальном тексте есть соответствующие инструкции. И что важнее всего — выводит текст с первой строки. 

Аналогичный результат можно получить через ''cat file.txt | less -R'', но уже без красивого экрана для чтения документации.

Но сила ''most'' в том, что она может показывать на одном экране произвольное число окон (каждое минимум 2 строки). В окнах могут быть разные файлы или один и тот же файл. У каждого окна свой отдельный режим: например, перенос строк или обрезка. Окна даже можно «связывать» — при прокрутке одного прокручиваются все связанные. 

===== Установить =====

''sudo apt install most''

===== Настроить =====

==== Выход по клавише [Q] ====

Выход из экрана most по клавише [Q]. Если в этот момент клавиатура переключена на другой язык, клавиша выхода не срабатывает.

Решение: создать файл для настроек ''most'' в каталоге профиля юзера и объявить дополнительную клавишу для выхода в русской раскладке:

''echo 'setkey exit "й"' > $HOME/.mostrc''

==== Открыть один файл в двух окнах ====

Открыть файл:

''most file.txt''

или

''file.txt | most''

Нажать последовательно “''Ctrl+X, 2''”, и выполнять переключение между окнами по клавише [O]. 

==== Открыть два разных файла ====

Для этого надо вызвать ''most'' с перечнем нужных файлов. Их может быть больше, чем два.

Взять названия файлов (вместе с путем к нему) в одинарные кавычки, чтобы не экранировать вероятные пробелы в названии каталогов/файлов.

''most 'installed Fonts.txt' 'Книги/readme.txt''' 

Будет открыт первый файл.

“''Ctrl+X, 2''”, переключиться во второе окно по клавише [O]. 

'':''
 
В низу окна появится предложение выбрать следующее действие.

''n''

В низу окна появится название второго файла. Ввод. Во втором окне появится содержимое второго файла.
