====== Cadence ======

[ @debian @install @sound @cadence ]

Cadence (каденция, муштра) — приложение/набор инструментов для управления звуковыми системами в Linux, которое обеспечивает удобный графический интерфейс для настройки JACK-аудиосервера, ALSA, PulseAudio и PipeWire.

Программа позволяет легко запускать/останавливать сервер, настраивать параметры (частота дискретизации, буфер) и управлять маршрутизацией звука, что полезно для профессиональной аудиообработки. 

См. https://kx.studio/Applications:Cadence

Установка Cadence в Debian очень нетривиальна, см. [[Debian:Install:Cadence]]

Вероятно, проще 

''sudo aptitude install jackd2''
