====== KeePassXC ======

[ @KeePassXC ]

Лучший open-source password manager. Форк умершего “KeePass”. Работает и под Windows.

Для андроид есть ''Keepass2Android'' — если положить файл с базой в Dropbox, то обновление файла будет происходить на лету. При возникновении нестыковок — записывает изменения в новую копию файла в Dropbox.

Надёжный, сбоев ещё не было.

===== Добавить новое поле =====

Предположим, что в форме логина есть три поля, которые надо заполнять:
	* название компании. Условно «MelkoOsft», а назовём его «CompanyId»
	* логин юзера
	* пароль юзера

Чтобы автоматически заполнялись все поля, надо сперва предопределить текстовые поля, которые расширение для браузера будет распознавать.

==== В KeepassXC ====

1. Даблклик по записи, в которой должно быть дополнительное поле, которое будет идентифицировано как «Строковое поле #1» (можно создать дохрена дополнительных полей).
2. Дополнительные 
	> Дополнительные атрибуты 
	> Создать
		Названия дополнительных полей надо выбирать самостоятельно. Я его назову «CompanyId».
3. Название нового поля должно выглядеть так: «KPH: [CompanyId]»
4. В правом поле вписать значение, которое должно быть вписано в настраиваемое поле (например, «MelkoOsft»).

Если есть другие логины, которые будут использоваться на той же странице — для каждого логина добавить своё дополнительное строковое поле.

==== В браузере ====

1. Открыть страницу с текстовыми полями, в которые надо вводить логин/пароль.
2. В расширении для браузера KeepassXC выбрать «Выбрать пользовательские поля входа» ("Choose custom login fields").
	a. В кнопках расширения кликнуть по [Имя пользователя] > кликнуть по полю, в котором надо вписывать имя пользователя.
	b. В кнопках расширения кликнуть по [Пароль] > кликнуть по полю, в котором надо вписывать пароль пользователя.
	c. В кнопках расширения кликнуть по [Строковые поля] > кликнуть по полю, в котором надо вписывать дополнительную информацию («Строковое поле #1»).
3. В кнопках расширения кликнуть [Подтвердить] — кнопки настройки исчезнут.
4. Перезагрузить страницу.
