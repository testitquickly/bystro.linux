====== IDE ======

[ @python @development @ide ]

Интегрированная среда разработки — IDE.

Вопрос привычек, удобства и требуемых возможностей. 

Вообще будет работать почти всё, что на глаза попадётся:
* “Eclipse” — древний, но работает
* “Eric” (есть и в репозитории Debian) — полностью написана на Python, есть множество инструментов для просмотра статистики кода, для анализа кода, для генерирования документации по проекту в html…
* “PyCharm” — IDE от JetBrains. Поддержка виртуальных окружений, БД, линтеры и форматтеры (pylint, mypy, black, ruff и др.), поддержка Jupyter Notebook. Есть PyCharm Community (бесплатное) и PyCharm Professional (платное 
* “[[Sublime]]” — блокнот с возможностью перестроить его как полноценное IDE
* [[KDevelop]] — нативное KDE для C, окружение для Python доступно через плагин: ''sudo apt install kdevelop kdevelop-python'' и при создании нового проекта надо сперва выбрать тип проекта ’Python’. Встроенный линтер — ''pycodestyle'', и его можно или включить (этого, в принципе, достаточно), или выключить. Все остальные доступны только через консоль.
