====== IDE ======

[ @python @development @ide @vscode @pyCharm @KDevelop @eric @eclipse @sublime @kate @vim ]

Интегрированная среда разработки — IDE.

* [[VSCodium]]

Форк VS Code (Visual Studio Code), легковесной IDE от Microsoft. Integrated Git, ИИ (через плагин), очень много плагинов. В управлении и настройках наследует подход Sublime. В базе работает незаметно-шустро, если не перевесить плагинами. Кроссплатформенное, потому что Electron.

* **PyCharm**

IDE от JetBrains. На Java. Есть PyCharm Community (бесплатное) — venv, БД, линтеры и форматтеры (pylint, mypy, black, ruff и др.), Jupyter Notebook; и PyCharm Professional (платное, подписка) — то же, + поддержка фреймворков (Django, Flask) и AI. Мощно, дорого, жрет ресурсы.

* [[KDevelop]]

Нативное, и поэтому шустрое KDE-software для программистов на C; окружение для Python доступно через плагин — ''sudo apt install kdevelop kdevelop-python'' — при создании нового проекта надо сперва выбрать тип проекта ’Python’. Встроенный линтер только один, ''pycodestyle'', и его можно или включить (этого, в принципе, достаточно), или выключить, все остальные линтеры можно использовать только через консоль как внешние утилиты. И вообще почти все инструменты для разработки в KDevelop недоступны, только консоль (лучше внешняя, внутренняя живет отдельной жизнью). Старик стабильно тащщит, но его последний релиз был в 2020-м году и нового не будет.

* **Eric**

Полностью на Python, есть встроенные инструменты для просмотра статистики кода, для анализа кода, встроенный генератор документации по проекту в html… Выглядит outdated, стремительно устаревает.

* **Eclipse**

Древняя тема, на Java, но работает. Настроек больше, чем параметров в обозримой вселенной. Ресурсожрателен. Когда-то был стандартом в индустрии, сегодня смотрится странно.

* [[Sublime]]

Текстовый редактор. Без плагинов — тупо текстовый редактор, с плагинами — шатл «Буран». Шустр. Управление/настройка как у VS Code. В умелых руках — см. [[Sublime:Как сделать Python-проект в Sublime]]

* **Kate**

Стандартный текстовый редактор в KDE. Превращается в IDE для Python после установки плагина ''kate-lsp'' (Language Server Protocol Client), которое приносит автодополнение, линтинг, переходы, подсказки и другие IDE-функции. Шустр, бо нативен. Встроенные возможности небольшие, но встроенная консоль делает всё.

* **vim**

Текстовый, кагбэ, редактор. Всегда готов ассимилировать ценителей непереходящих ценностей. Resistance is futile.
