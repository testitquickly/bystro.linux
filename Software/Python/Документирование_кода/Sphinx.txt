====== Sphinx ======

[ @python @doc @sphinx ]

Установить:

''pipx install sphinx sphinx-autodoc''

Инициализация:

''sphinx-quickstart''

В ''conf.py'' добавить 'sphinx.ext.autodoc', указать sys.path.

Сборка:

''make html''
