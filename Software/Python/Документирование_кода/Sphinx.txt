====== Sphinx ======

[ @python @doc @sphinx ]

Установить глобально:

''pipx install sphinx''

Перейти в каталог с файлами RST. 

Убедиться в том, что в корне этого каталога НЕТ файла ''index.rst''. Если есть — удалить или переименовать, потому что Sphinx создаст новый.

''.. toctree::''
   '':maxdepth: 2''
   '':caption: Contents:''

   ''bystro.linux/index''

Начать работу:

''sphinx-quickstart''

Начнется процесс инициализации фреймворка. На первый вопрос «Разделить каталоги исходных файлов и результатов сборки?» ответить ''y'', дальше по своему усмотрению.

Появятся новые файлы:

source/
├── conf.py
├── index.rst
└── _build/

Перейти в катаог ''source/'' и выполнить:

''sphinx-build -b html . _build/html''

или

''make html''

Появится ''source/_build/html/index.html''

Открыть его в браузере.
