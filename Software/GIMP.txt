===== Настройка шаблонов =====

Легко создать новый шаблон, но непонятно, как удалить ненужные, и непонятно, как сохранять в новом шаблоне прозрачную основу. Оказывается, это надо делать вручную.

Сперва надо создать свой шаблон:

Создать новое изображение: 

''File'' 
''> Create Template'' 

Указать имя шаблона.

Из GIMP можно выйти.

Отредактировать файл с шаблонами:

''mcedit $HOME/.config/GIMP/''__''2.10''__''/templaterc''

Найти свой шаблон. Отредактировать его параметр: 

''(fill-type transparent)''

Сохранить, выйти.

Удалять/редактировать другие шаблоны нет смысла, этот файл будет перезаписан каждый раз, когда GIMP будет погашен.

==== Создать файл из шаблона ====

''File''
''> New''
''> Template''

Выбрать нужный шаблон.
