====== Todoman ======

[ @console @todolist @todoman ]

Младший брат [[Taskwarrior]], написан на [[Python]]

Все задачи сохраняются в отдельных файлах iCalendar, что позволяет синхронизировать их с другими компьютерами по протоколу CalDAV, например, через ''vdirsyncer''. Каждый список задач представлено отдельным каталогом.

===== Установить =====

* [[Debian:Install:Todoman]]

===== Основные команды =====

todoman list [OPTIONS] [LISTS]...
todoman new [OPTIONS] [SUMMARY]...
todoman cancel [OPTIONS] TODOS...
todoman copy [OPTIONS] IDS...
todoman delete [OPTIONS] IDS...
todoman done [OPTIONS] TODOS...
todoman edit [OPTIONS] ID
todoman flush [OPTIONS] - Delete done tasks. This will also clear the cache to reset task IDs.
todoman move [OPTIONS] IDS...
todoman show [OPTIONS] ID
todo list --sort due,-priority
The tasks can be sorted with the --sort argument. Sorting may be done according to the following fields:
	description
	location
	status
	summary
	uid
	rrule
	percent_complete
	priority
	sequence
	categories
	completed_at
	created_at
	dtstamp
	start
	due
	last_modified
