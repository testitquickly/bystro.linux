====== Todoman ======

[ @console @todolist @todoman ]

Младший брат [[Taskwarrior]], написан на [[Python]]

Все задачи сохраняются в отдельных файлах iCalendar, что странно, но позволяет синхронизировать их с другими компьютерами по протоколу CalDAV, например, через ''vdirsyncer''. 

Каждый список задач представлен отдельным каталогом внутри основного каталога (см. конфиг при установке). Нужен новый список — сперва вручную сделать новый подкаталог с соответствующим названием.

===== Установить =====

* [[Debian:Install:Todoman]]

===== Основные команды =====

todoman или todoman list — вывод списка задач
todoman new 
todoman cancel
todoman copy
todoman delete
todoman done
todoman edit
todoman flush — удаляет выполненные задачи, чистит кэш и сбрасывает task IDs.
todoman move
todo list --sort due,-priority
Опции сортировки задач:
	description
	location
	status
	summary
	uid
	rrule
	percent_complete
	priority
	sequence
	categories
	completed_at
	created_at
	dtstamp
	start
	due
	last_modified
