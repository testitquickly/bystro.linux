====== Baloo ======

[ @search ]

Установлен по-умолчанию, глубоко интегрирован в систему, отлично работает с [[KRunner]]. 

Однако его index и способы поиска устроены очень невнятно, местами непредсказуемо. Бывают и баги. Самые заметные проблемы начинаются, когда он внезапно «просыпается» и начинает индексировать — он берёт у системы очень много ресурсов, нагрузка на CPU огромна, а размер базы индексируемых файлов может неоправданно увеличиться в размерах в десятки раз.

Есть обоснованное мнение о том, что Baloo следует использовать только для поиска файлов, а для поиска по содержимому файлов лучше использовать [[Recoll]]. Я для поиска файлов использую древний [[find]]. Ещё есть новый [[FSearch]].

===== Конфиг =====

''mcedit ~/.config/baloofilerc''

===== Проверка индекса =====

''balooctl status''

===== Перезапуск индекса =====

''balooctl purge''
