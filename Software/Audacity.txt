===== Метки =====

Можно добавлять в место курсора, когда проигрывание на паузе. По-умолчанию [Ctrl+B], но удобнее назначить на [Alt+2].

	Можно добавить метку к месту курсора во время проигрывания трека (Ctrl+M). Неудобно то, что трек в момент набора текста метки продолжает проигрываться.

Можно сперва выделить часть трека и добавить метку. Также уже существующие «точечные» метки можно ухватывать курсором мыши и растягивать.

Есть возможность добавить метку с текстом из буфера обмена. Удобно при редактировании по уже существующему скрипту, из которого можно копи-пастить, или когда надо разбить на части длинную запись с винила.

1. Разметить весь трек метками
2. Файл > Экспорт > Экспорт в несколько файлов (или Ctrl + Shift + L по-умолчанию)
3. Выбрать папку и формат файлов для экспорта.
4. Разделение файлов по принципу > Метки
5. Имена файлов > Используется метка/имя трека
6. Экспорт

Каждый помеченный раздел будет сохранен как отдельный файл.

===== Нормализация =====

* Убрать смещение по оси амплитуды
* Нормализовать пик амплитуды до -4.0 dB

===== Компрессия =====

* Порог: -12 dB
* Нижний порог шумов: -40 dB
* Пропорции: 5:1
* Время атаки: 0,20 сек
* Время затухания: 1,0 сек
* Компенсация усиления до 0 dB после сжатия = Да

Настраивать в каждом случае отдельно.

===== Подавление шума =====

Во время записи надо помолчать секунд пять при включенном микрофоне, чтобы записать общий фоновый шум. 

1. При редактуре выделить эти секунды
2. Эффекты > Подавление шума > Получить профиль шума.
3. Выделить всю дорожку (Tab).
4. Эффекты > Подавление шума > Настроить:
	1. Подавление шума: 12 dB
	2. Чувствительность: 6.00
	3. Сглаживание частоты: 3
	4. Шум: уменьшить
5. Ок

===== Нормализация громкости =====

Воспринимаемая громкость до -23,0 LUFS

===== Приглушение музыки =====

В англ. „Auto Duck“.

Эффект сложный, надо научиться использовать его грамотно, делать очень явные, ощутимые, «длинные» и логические паузы в речи, чтобы автоматика успевала подхватить «молчание», повысить уровень звука, а затем надо бодро начинать говорить, и постоянно говорить какое-то время, чтобы автоматика «приглушила» музыку, и чтобы восприятие слушателя приняло эти перепады адекватно. Если говорить в обычном режиме, с короткими паузами, то эффект от импульсивного приглушения/усиления музыки будет ощущаться как издевательство над слухом. 

1. Добавить в проект трек с мелодией. Передвинуть его НАД треком с голосом.
2. Выделить всё содержимое трека с музыкой.
3. Выбрать Эффекты > Приглушение музыки
4. Для каждого микрофона и голоса эти настройки будут разными, но в принципе можно начать с уровня -17dB.
5. Длина внешнего затухания — пусть остается 0.5 секунд.
6. Длина внешнего нарастания — 0,75 секунд.

Если просто нужна ненавязчивая фоновая музыку, то лучше поставить её отдельной дорожкой, приглушить всю. Также она должна быть ритмично закольцованной, без переходов и изменений, иначе мозг начнёт пытаться прослушать музыку сквозь голос, и эффект будет обратный. 

===== Анализ спектра аудио =====

Анализатор спектра показывает график всех частот, присутствующих в звукозаписи. 

Более темные области — это области с очень низкой интенсивностью частот.

Оранжевые и желтые области представляют собой частоты с высокой интенсивностью звука. 

Запись скрипки, в частности, ясно демонстрирует богатое гармоническое содержание каждой сыгранной ноты (это проявляется на спектрограмме в виде нескольких более высоких частот, генерируемых для каждой основной частоты). Это контрастирует с записью свиста, которая имеет очень сильную основную составляющую и имеет только одну дополнительную гармонику, что указывает на то, что человеческий свист очень близок к чистой синусоиде.
