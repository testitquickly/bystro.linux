====== EdrawMind ======

[ @edraw_mind @mind_map ]

Единственный майнд-манагер для Linux, наиболее похожий по функциональности на MindManager от MindJet (тот только для Win и Mac). Можно адекватно настроить внешний вид майндмапы и упорядочить направление веток.

В бесплатной версии есть мощное ограничение — не больше 100 топиков в одной майндмапе.

===== Установить =====

* [[Debian:Install:EdrawMind]]

===== Настроить =====

Служебные файлы будут помещены в каталог ''~/Edraw'', поменять это невозможно.

Доступные языки: английский и другие, русского нет.

Переключение на тёмную тему: Настройки > справа вверху под иконкой аккаунта есть иконка Skin.

===== Шорткаты =====

Есть смысл очень детально пересмотреть всё, что доступно. Некоторые шорткаты явно наследуют MindManager.

Переключиться в режим Outline (текстовый список) = F10

Сфокусироваться на определенной ветке = F3

Оставить на экране только ветку в фокусе (Drill Down) = F4

Display branch = Alt + D

Сделать топик задачей — поставить фокус на топике, вызвать панель задач (справа) через Advanced > Task, на этой панели тыкнуть по иконке [+] — станет доступной форма настроек задачи. После назначения ресурса на задачу и прочих настроек можно вызвать диаграмму Гантта и расположить все задачи в нужной последовательности.

Режим Slideshow > Брэйнсторминг (набросать идеи одну за другой, затем каждую идею можно перетащить на майндмапу) = F8

===== Сложности =====

В окне Open File темный фон, названия файлов нечитаемые. Интуитивно обходится через Ctrl+A.

Приложение генерирует много процессов, поэтому на некоторых компьютерах процессор может запыхтеть.
