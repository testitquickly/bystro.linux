====== MC и кодировка win1251 ======

[ @mc @utf8 @win1251 ]

===== Как сделать читаемым файл с кодировкой win1251 =====

1. Убедись в том, что основная кодировка в MC установлена на UTF-8

F9 → Настройки → Биты символов = UTF-8 и есть галочка на "Полный 8-битный вывод".

2. Убедись в том, что файлы будут открываться на редактирование внутренним редактором:

F9 → Настройки → Конфигурация → Прочие настройки:
- Встроенный редактор
- Встроенный просмотр

3. Открыть по F4 файл с текстом в виндовс-кодировке:

F9 → Команда → Выбор кодировки → Windows 1251

F9 → Настройки → Сохранить настройки

Это глобальная настройка, поэтому произойдёт следующее: 
* отныне ВСЕ файлы будут открываться на просмотр/редактирование через MC в кодировке Win1251. 
* конец строки в таких файлах будет отображаться символом '^M' (несколько раздражает, но ничего не поделать).

А если попадётся файл в кодировке UTF-8, то снова будут кракозябры, только уже другого рисунка. Придётся снова «F9 → Команда → Выбор кодировки → выбрать "Без перекодировки"». 

===== Как открыть тот же каталог в соседней панели =====

''Alt+I'' opens the active panel's directory on the other panel.

''Alt+O'' opens the directory your cursor is on.
