====== Qalculate! ======

[ @console @qalculate ]

===== Установить =====

''sudo apt install qalculate-qt''

===== Настроить =====

Важно настроить его правильно, чтобы в будущем не страдать от ерундовых отклонений в цифрах.

Режим
Основание системы счисления = Десятичное
Отображение чисел 
	= Обычное
	= Показывать конечные нули
	= Округлять половинные числа вверх
	= Экспоненциальная форма комплексных чисел

==== Результат вычислений ====

''Параметры''
''> Числа и операторы''
''> Режим отображения интервалов'' = Плюс/Минус

''Параметры''
''> Разбор и вычисление''
''> Статус выражения'' = Выкл

Иначе условное ''10.41*20'' вместо ''= 208,2'' будет отображаться как нечто абстрактное.
