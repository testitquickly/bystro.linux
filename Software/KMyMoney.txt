====== KMyMoney ======

[ @finance @k_my_money ]

Адекватная, старинная аппликуха для учёта финансов. Learning curve может быть тяжеловата. Аналогов море.

===== Установка =====

* [[Debian:Install:KMyMoney]]

===== Зашифровать файл ключом GPG =====

Если надо защитить файл ВООБЩЕ, то надо использовать что-то вроде криптоконтейнера VeraCrypt — будет создан файл-контейнер, в котором будет спрятан файл *.kmy, а снаружи будет длиннопарольная защита или даже ключ с внешнего usb-диска или даже подпись gpg-ключом. Или всё сразу.

В быту удобнее сделать защиту файла только через шифрование с помощью личного ключа GPG — это превращает содержимое файла *.kmy в нечитаемую абракадабру. Зашифрованный файл можно безопасно положить в Dropbox или даже отослать по почте — даже если учётную запись в Dropbox взломают, без gpg-ключа и его пароля вскрыть базу получится только паяльником внутри владельца файла.

Надо учесть, что это защитит только от вороства, когда злой умышленник заберёт файл *.kmy и захочет открыть его на своём компьютере. На моём ноутбуке этот же зашифрованный файл *.kmy в определённых обстоятельствах будет открываться СРАЗУ, безо всяких паролей, потому что gpg-ключ тут прописан в системе, и ему присвоен высочайший уровнь доверия. В определенных обстоятельствах уже открытый файл может быть открыт заново. 

Если шифрования с gpg достаточно:
1. сделать новый ключ [[GPG]] (или взять существующий и зарегистрировать его в системе)
2. прописать его в настройках KMyMoney
''> Настройка''
''> Настроить KMyMoney''
''> Расширения (Plugins)''
''> XML Storage («Сохранение XML»)''
''> настройки данного плагина''
''> Use GPG Encryption''
''> выбрать из выпадающего списка нужный ключ из существующих в системе (если их несколько)''
''> [ОК]''
''> Файл''
''> Сохранить (подтвердить понимание риска потери данных если ключ потеряется)''
''> Файл''
''> Выход''
3. Заново открыть файл *kmy

Зашифрованный файл *.kmy 
* можно перемещать или переименовывать,
* невозможно открыть в другой системе без предварительного провисывания в систему того же gpg-ключа, которым он был зашифрован.

===== Настроить сочетания клавиш =====

''Настройка''
''> Комбинации клавиш''

* Дублировать: Ctrl+D
* Калькулятор: Ctrl+K
* Отменить: Ctrl+Z
* Создать: Ctrl+N
* Сохранить: Ctrl+S
