====== cmus ======

[ @console @audio ]

Консольный аудиоплеер.

===== Установить =====

''sudo apt install cmus''

===== Интерфейс =====

* Ctrl+5 — браузер по файлам на hdd
* Ctrl+7 — справка по клавишам (их много)

===== Выход =====

''q'', ''y''
