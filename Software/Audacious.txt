====== Audacious ======

@sound @audacious
--------------------

Внешне может выглядеть как Winamp или [[qmmp]]. 

===== Настроить модуль вывода звука =====

После установки по-умолчанию используется первый модуль вывода в списке — PipeWire. У меня PulseAudio.

''Настройки''
''> Звук''
''> Настройки вывода''
''> Модуль вывода = Вывод PulseAudio''
''> Разрядность = Автоматически''

===== Настроить скорость воспроизведения =====

''Настройки''
''> Модули''
''> Эффект''
''> Скорость и высота''
''> Настройки эффекта''
''> Разделение по частоте'' 
''> Множитель = 1,35''

Если на повышенной скорости голос слишком «визгливый», можно уменшить число для «Частота» (автоматически изменится значение для «Множитель»).

Если что-то пошло не так, можно перебить «Частота» на ноль и поискать более подходящее значение.

===== Настроить «Глобальные клавиши» =====

''Настройки''
''> Модули''
''> Общие''
''> Глобальные горячие клавиши''

Дальше по своим предпочтениям.
