====== Каталоги в --exclude ======

[ @rsync @exclude @folder ]

===== Относительный путь к исключаемым каталогам =====

Надо перенести содержимое ''/Документы'' в ''//NAS/Документы'', кроме содержимого нескольких каталогов:

''rsync -auzv \''
	''-e "ssh -i $ssh_key" \''
	''--remove-source-files \''
	''--skip-compress=$file_list \''
	__''--exclude="/home/hub/Документы/Mind Maps/" \''__
	__''--exclude="/home/hub/Документы/Автомобиль/_CarteaVerde/" \''__
	''"/home/hub/Документы/" "$nas_enc_docs"''

Но в этом случае всё содержимое каталогов, которые //должны// быть проигнорированы, ВНЕЗАПНО будет полностью перенесено в NAS. Есть неочевидный момент — RSync рассматривает только относительные пути к каталогам под параметром ''--exclude'', поэтому перечисленные тут каталоги RSync //видит// как ''/home/hub/Документы//home/hub/Документы/Mind Maps/'', а такого каталога нет, поэтому команда воспринимается как корректно отработанная и скрипт продолжает работу…

Правильно так: 

''rsync -auzv \''
	''-e "ssh -i $ssh_key" \''
	''--remove-source-files \''
	''--skip-compress=$file_list \''
	__''--exclude="Mind Maps/" \''__
	__''--exclude="Автомобиль/_CarteaVerde/" \''__
	''"/home/hub/Документы/" "$nas_enc_docs"''

В этом случае RSync //увидит// каталог ''/home/hub/Документы/Mind Maps/'' и послушно проигнорирует его.

Ещё надо учесть, что каталог (со всем своим содержимым) в exclude надо добавлять с закрывающим слэшем, тогда RSync проигнорирует и каталог, и всё его содержимое. 

''--exclude="Mind Maps/"''

Если закрывающего слэша не будет, RSync проигнорирует сам каталог, но все файлы или подкаталоги внутри него будут включены в синхронизацию.

===== Проверить наличие каталогов =====

Учесть ещё одну особенность RSync — если какой-то из каталогов в ''--exclude'' исчезнет с диска, сообщений об этом не будет. 

Можно обработать эту ситуацию так:

{{{code: lang="sh" linenumbers="True"
	# каталоги для исключения из /Документы/
	# без слэшей
docs_excluded_list=(
	"Mind Maps"
	"Автомобиль/_CarteaVerde1"
)

	# создать пустой массив (список каталогов) для этого раздела
docs_excluded=()

	# добавить в созданный массив элементы из списка в цикле, с проверкой "каталог есть на диске", иначе полный стоп
for dir in "${docs_excluded_list[@]}"; do
    if [[ -d "$docs/$dir" ]]; then
        docs_excluded+=(--exclude="$dir/")
    else
            # показать алерт
        echo -e "\n\t${clr_RED_BRIGHT}Стоп! Отсутствует каталог '$docs/$dir' ${clr_NONE}\n" >&2
            # прекратить выполнение скрипта после сообщения об ошибке
        exit 1
    fi
done

rsync -auzv \
	-e "ssh -i $ssh_key" \
	--remove-source-files \
	--skip-compress=$file_list \
	"${docs_excluded[@]}" \
	"$docs" "$nas_docs_enc"
}}}

Для следующей команды синхронизации надо будет прописать отдельную переменную и повторить всю эту структуру.
