===== Автозапуск Dropbox при старте системы =====

Иногда всё работает само по себе. Если есть сложности, смотрим далее.

==== Автозапуск только демона ====

Не будет иконки в трее. Не очень удобно, но работает.

'''
Параметры системы
> Рабочая среда 
> Запуск и завершение 
> Автозапуск 
> [+Добавить]
> [Добавить сценарий входа в систему]
'''

вписать в поле «Имя» команду

''~/.dropbox-dist/dropboxd''

Проверить, если Dropbox запущен:

''~/.dropbox-dist/dropbox.py status''

==== Автозапуск с иконкой в трее ====

Создать скрипт 

''mcedit ~/.dropbox_start.sh''

Вставить:

{{{code: lang="sh" linenumbers="True"
#!/bin/bash

    # Остановить Dropbox демон (на всякий случай)
killall dropbox

    # Запустить демон Dropbox
$HOME/.dropbox-dist/dropboxd

    # Запустить gui Dropbox (иконка в трее)
$HOME/.dropbox-dist/dropbox.py start

}}}

Сделать файл выполняемым:

'''
chmod +x ~/.dropbox_start.sh
'''

Добавить его в автозапуск

''Параметры системы''
''> Рабочая среда'' 
''> Запуск и завершение'' 
''> Автозапуск''
''> + Добавить…''
''> Добавить сценарий входа в систему''
