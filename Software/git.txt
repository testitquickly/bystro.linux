===== Если файлы обновлены в режиме 'file mode' =====

Иногда при перемещении файлов проекта git может перечитать их обновленные свойства через “''chmod -R 777''” как появление новых файлов в репозитории вообще, и предложит перекоммитить почти всё.

Решение: в каталоге с файлами и ‘.git‘ выполнить:

''git config core.fileMode false''

===== Посмотреть =====

''git status''

Кратко:

''git status -sb''

Последние пять коммитов:

''git reflog -5''

''git log -5''

Компактный лог (всё):

''git log --oneline --graph --decorate --all''

Компактный лог (последние 5):

''git log --oneline --graph --decorate -5''

Список файлов:

''git ls-files''

Ветки и ссылки:

''git branch''

''git branch -a''

Поиск

''git grep text''

''git log --grep="fix"''

''git log -- file''

Конфиг:

''git config --list''

''git config --global --list''

Статистика:

''git shortlog -sn''

''git count-objects -v''

''git log --stat''
