====== git ======

[ @console @smartgit @git @github ]

Скорее всего, он уже установлен. Проверка:

''git version''

===== Установить =====

Можно установить git в ПОЛНОМ комплекте:

''sudo apt-get install git-all && git version''

Для постоянной работы будет удобно использовать GUI-клиент (мне нравится [[SmartGit]]). Также ничего не запрещает нативно работать с git в консоли или в [[IDE]].

===== Если файлы обновлены в режиме 'file mode' =====

Иногда при перемещении файлов проекта git может перечитать их обновленные свойства через “''chmod -R 777''” как появление новых файлов в репозитории вообще, и предложит перекоммитить почти всё.

Решение: в каталоге с файлами и ‘.git‘ выполнить:

''git config core.fileMode false''

===== Посмотреть =====

''git status''

Кратко:

''git status -sb''

Последние пять коммитов:

''git reflog -5''

''git log -5''

Компактный лог (всё):

''git log --oneline --graph --decorate --all''

Компактный лог (последние 5):

''git log --oneline --graph --decorate -5''

Список файлов:

''git ls-files''

Ветки и ссылки:

''git branch''

''git branch -a''

Поиск

''git grep text''

''git log --grep="fix"''

''git log -- file''

Конфиг:

''git config --list''

''git config --global --list''

Статистика:

''git shortlog -sn''

''git count-objects -v''

''git log --stat''

===== Игнорировать файлы/каталоги =====

Сделать файл ''.gitignore'', в котором надо написать

''# каталоги''

''# кеши / временные файлы''
''*.swp''
''*.swo''
''*~''
''*.bak''
''*.tmp''
''*.kate-swp''

''# системные файлы''
''Thumbs.db''

''# логи''
''*.log''

''# телефонные книги''
''*.vcf''

Записи из этого файла действуют ДО того, как каталоги и файлы, которые надо игнорировать, будут добавлены в репозиторий. Если надо игнорировать уже существующий каталог, надо сперва удалить его из учета. А путь к нему зависит от места, в котором лежит файл .gitignore — в корне репозитория или глубже по иерархии. Если в корне, тогда надо указать полный путь до игнорируемого каталога, начиная с каталога, в котором находится репозиторий:

''git rm -r --cached'' __''полный путь до игнорируемого каталога от корня репозитория/''__

После этого закоммитить изменения:

''git commit -m "Stop tracking folder…"''

И добавить каталог в .gitignore:

__''полный путь до игнорируемого каталога от корня репозитория/''__
