====== bluetoothctl ======

[ @console @bluetooth @fzf ]

Можно управлять подключениями BT из консоли.

''bluetoothctl''

Откроется интерактивная оболочка. ''help'' покажет доступные команды.

У меня Bluetooth на ноутбуке постоянно включен, но если нет, то:

''power on''

Поиск устройств, если надо соединить новое:

''pairable on''
''discoverable on''
''scan on''

Глянуть список устройств:

''devices''

Например:

//Device 20:74:CF:44:ED:05 Aeropex by AfterShokz//
//Device C4:6E:7B:6C:20:7A VMK25//

''scan off''

Подключиться к первому. Название устройства подхватывается через [Tab]. 

''pair 20:74:CF:44:ED:05''
''trust 20:74:CF:44:ED:05''
''connect 20:74:CF:44:ED:05''

Выход из интерактивной оболочки:

''exit''

===== Удобное использование =====

Чаще всего надо просто подключить устройство, которое уже было ранее сопряжено с ноутбуком, поэтому рассмотрим только этот сценарий:

* включить ''bluetoothctl''
* получить список уже ранее сопряженных устройств
* подключить одно из них по его “mac-address”

Понадобится fzf — интерактивный выбор из текста в терминале.

''sudo apt install fzf''

В ~/.bashrc сделать алиас:

''alias bt='bluetoothctl devices | fzf | awk "{print \$2}" | xargs -r bluetoothctl connect'''

Теперь по команде ''bt'' в консоли появляется интерактивная оболочка fzf, в которой можно стрелками выбрать нужное устройство и нажать Enter — начнется его подключение.
