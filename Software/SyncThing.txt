====== SyncThing ======

[ @syncthing @files @nas ]

Для того, чтобы изредка скидывать файлы в NAS, SyncThing — бесконечно избыточное решение, там достаточно [[RSync]]. 

SyncThing нужен, когда есть //множество// устройств, между которыми надо часто (постоянно) синхронизировать файлы. 
 
В настройке штука замороченная.

===== Установка =====

* [[Debian:Install:SyncThing]]

===== Запуск =====

В консоли

''syncthing''

В логе мелькнет адрес, по которому на локахосте работает сервер (пример):

''Access the GUI via the following URL: http://127.0.0.1:8384/''

Также  должен запуститься основной браузер с этим адресом.

Сервис будет работать в фоне. Его можно захлопнуть или через браузер, или прибить процесс.

===== Настройка =====

==== Аккаунта ====

Логично с самого начала установить имя пользователя и пароль, если на ноуте несколько пользователей. Да и если только один.

==== Каталога на устройстве ====

По-умолчанию предоставлен каталог ''~/sync'', но надо создавать/открывать свои.

''Папки'' 
''> Добавить папку''

Указать полный путь к ней.

==== Каталога на удаленном устройстве (NAS) ====

''Удалённые устройства'' 
''> Добавить удалённое устройство''

===== Подключение =====

''Действия'' 
''> Показать ID''
