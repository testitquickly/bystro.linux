===== Установить =====

[[Debian:Install:Krusader]]

===== Файл с закладками =====

Закладки Krusader хранятся в файле ''~/.local/share/krusader/krbookmarks.xml''

Есть смысл этот файл регулярно бэкапить и использовать при переустановках.

Можно даже переместить его в какое-то общее хранилище, чтобы был на виду, а вместо него поставить “мягкую“ ссылку на перемещённый файл — HowTo:Как сделать symlink

===== Отображение расширения файла =====

По-умолчанию расширения отображаются в отдельной колонке. Чтобы отображать их рядом с именем файла, надо сделать правый клик по заголовку колонки и убрать галочку «Расширение».

===== Редактировать только имя файла =====

''Настройка''
''> Настроить Krusader''
''> Панель''
''> Основные''
''> Операция''

Снять галочку «Выделять расширение при переименовывании».

===== KRename =====

Если есть много файлов, которые начинаются с нумерации вроде 01, 02, 03, то:

''> Расширенное имя файла''
''> Шаблон''

Использовать это:

''[$2;75]''

Читается как «Убрать первые два символа, оставить все остальные до 75».

Менять цифры по необходимости.
