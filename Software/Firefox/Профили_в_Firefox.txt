====== Профили в Firefox ======

[ @firefox @user @profile ]

Есть официальная по всему этому делу — https://support.mozilla.org/en-US/kb/profile-manager-create-remove-switch-firefox-profiles

===== Создать новый профиль =====

	Открыть url „about:profiles” 
	> [Создать новый профиль> 
	> дать ему внятное имя, например, “''deserviciu''”

Профили создаются в /~/.mozilla/firefox в виде каталогов с инопланетным названием. Если профиль надо будет перемещать на другие компьютеры, разумно указать для него отдельный каталог на диске. 

Каждый новый профиль помечается как «по-умолчанию», поэтому надо найти свой основной профиль и отметить его «as default». Обычно это профиль с названием «default-release».

===== Импортировать профиль =====

В новый профиль можно переместить файлы из уже существующего — должны подтянуться все настройки. Учесть, что будет скопировано всё, и куки и кэш. 

Можно просто скопировать всё с затиранием существующих файлов, но сперва надо выключить Firefox, иначе перезапись каких-то файлов будет запрещена и импорт будет некорректным.

===== Запуск браузера с выбором профиля =====

''firefox -P''

Откроется окно с перечнем профилей.

В окне выбора профилей можно отметить галочкой один из профилей, которые в будущем будет загружаться автоматически. Это аналог назначения профиля «as default» на странице „about:profiles”.

===== Запуск браузера с определенным профилем =====

Можно сделать ярлык для запуска определенного профиля Firefox:

''firefox -p deserviciu''

Если это рабочий аккаунт, то лучше назначить этот профиль дефолтным. 

Иногда безопаснее использовать для работы и для личного не разные профили, а разные браузеры.

===== Открывать вкладки в другом профиле Firefox =====

Установить расширение [[https://addons.mozilla.org/en-US/firefox/addon/open-with/|open-with]].

В настройках расширения перейти на «Кликните здесь для загрузки файла» — будет предложено сохранить файл „open_with_linux.py”.

В консоли сделать этот файл выполняемым:

''chmod u+x open_with_linux.py''

Выполнить этот файл с параметром "install":

''./open_with_linux.py install''

Вернуться на страницу настроек расширения и тыцнуть [Проверить установку> — ждём позитивный фидбэк.

[Поиск браузеров> > в списке найденных браузеров будет и Firefox (можно и вручную добавить) — открыть его настройки („Изменить”), переименовать и прописать команду открытия браузера с указанием нужного профиля такого вида:

''/usr/lib/firefox/firefox -p otherProfile''

Тыкнуть любому линку правой кнопкой — в самом низу появится новый пункт меню „Open With” с выбором браузеров/профилей, которые были прописаны в настройках.

В Firefox с другим профилем повторить эту процедуру установки расширения но уже без скачивания и настройки файла ''open_with_linux.py''.
