====== Отключить запрос пароля для sudo ======

[ @console @sudo @password ]

Можно (хотя и //не нужно//) отключить запрос пароля при выполнении команд через ''sudo'':

''sudo mcedit /etc/sudoers.d/nopasswd''

Вставить:

__''astenix''__ ''ALL=(ALL) NOPASSWD: ALL''

Сохранить и выйти.

При такой настройке система //ВООБЩЕ// не будет запрашивать пароль для любой задачи через sudo, и это вообще не здорово. Можно отключить запрос паролья только для отдельных команд, например, для обновления Debian:

__''astenix''__ ''ALL=(ALL) NOPASSWD: /usr/bin/apt update, /usr/bin/apt upgrade''

Или разрешить выполнять отдельный скрипт:

__''astenix''__ ''ALL=(ALL) NOPASSWD: /bin/bash /полный/путь/к/скрипту.sh''

Выставить этому файлу права:

''sudo chmod 440 /etc/sudoers.d/nopasswd''

Проверить синтаксис файла

''sudo visudo -c''

Ждём в ответ 

''/etc/sudoers.d/nopasswd: parsed OK''

иначе алярм.

Закрыть окно Konsole или просто открыть новое — настройки подхватятся моментально. 

Если уже пошли по столь радикальному пути, можно даже удалить файл с таймаутом:

''sudo rm /etc/sudoers.d/timeout''
