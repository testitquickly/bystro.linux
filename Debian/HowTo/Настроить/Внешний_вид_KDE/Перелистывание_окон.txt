====== Перелистывание окон ======

[ @kde @window @перелистывание ]

''Параметры системы'' 
''> Рабочая среда'' 
''> Диспетчер окон'' 
''> Переключение окон'' 
''> Основные''

	Показывать выбранное окно: Нет
	Визуализация: //Карусель// — самая впечатляющая, на деле нужно проще. //Modern Informative.//
	Сортировка: По времени последнего переключения на окно
	В настройках карусели:
		* Показывать заголовки окон
		* Масштаб: на второй риске
		* Анимировать переключение
		* Длительность анимации: 300 мс

Очень хороша «//Modern Informative//» — вертикальный текстовый список окон, чётко видны и иконка приложения, и название, без условного содержимого окон.

Можно перелистывать окна одного типа, например, только Firefox: ''Alt+`''

Это комбинация по-умолчанию в

''Параметры системы'' 
''> Ввод и вывод''
''> Клавиатура''
''> Комбинации клавиш''
''> Системные службы''
''> KWin''
''> «На одно окно вперёд текущего приложения»''

===== “Переключатель окон установлен неправильно” =====

Может случиться так, что при обновлении системы произошел неочевидный апгрейд Plasma, и настройки старой версии не состыковались с новой. Например, будут установлены modern_informative и fast_thumbnails — две несовместимые кастомные TabBox темы, которые требуют старого KDE QML API (org.kde.kwin 2.0), который в Plasma 6.3 отсутствует. Будет конфликт и сообщение о том, что “Переключатель окон установлен неправильно”.

Радикальный сброс настроек:

''sudo apt install kwin-addons qml6-module-qtquick-controls qml6-module-qtquick-layouts''

''rm -rf ~/.local/share/kwin/tabbox/*''

''kwriteconfig5 --file kwinrc --group TabBox --key LayoutName "thumbnail_grid"''

''kwin_x11 --replace &''
