====== Записать результат выполнения команды в файл ======

[ @console @main @file ]

Будет создан новый файл, в котором записан результат команды ls -R /home (перечислить всё содержимое каталога home):

''ls -R /home > home.txt''

Без указания направления вывода в файл вывод пойдёт в консоль (на экран). Если иноформации мало, то она будет читаема, а если много — удобнее скидывать сразу в файл.

===== Как сделать дозапись данных в файл =====

Надо использовать символы >> (два знака больше)

''ls -R /home >> home.txt''

Вся новая информация будет добавляться в конец существующего файла.
