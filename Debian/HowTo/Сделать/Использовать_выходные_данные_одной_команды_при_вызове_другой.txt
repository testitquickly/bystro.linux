====== Использовать выходные данные одной команды при вызове другой ======

[ @console ]

В этом примере появится файл, имя которого будет версия текущего ядра linux.

''touch $(uname -r)''

**Пояснение**: команде ''touch'' передается с помощью конструкции $() команда ''uname -r'' выходные данные которой можно использовать для аргумента первой команды.
