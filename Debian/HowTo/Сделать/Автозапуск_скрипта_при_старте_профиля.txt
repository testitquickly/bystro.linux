====== Автозапуск скрипта при старте профиля ======

Положим, есть исполняемый скрипт в *.sh и он должен автоматически выполняться при старте профиля пользователя.

Можно сделать каталог

''mkdir ~/.config/autostart-scripts''

и положить [[Симлинки и хардлинки:symlink]] на этот файл в 

''cd ~/.config/autostart-scripts && ln -s /…script.sh''

Дальше произойдет неочевидное:

* в каталоге ''~/.config/autostart/'' появится файл ''*.desktop'' со ссылкой на ''/…script.sh''
* каталог ''~/.config/autostart-scripts'' будет удален
* перечень всех файлов ''~/.config/autostart/*.desktop'' появится в Параметры системы > Автозапуск
* при загрузке профиля выполняются все скрипты из файлов ''*.desktop'' из каталога ''~/.config/autostart/''
