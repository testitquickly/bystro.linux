====== Сортировка строк в файле ======

[ @console @sort ]

Есть файл text.txt с каким-то набором строк.

===== Отсортировать строки в файле по алфавиту =====

''sort text.txt -o text.txt''

===== Отсортировать строки и сохранить результат в другом файле =====

''sort text.txt > text_sorted.txt''

===== Отсортировать с учетом регистра =====

По умолчанию, sort учитывает регистр букв (Linux way). 

Чтобы сортировать строки БЕЗ учета прописных и заглавных букв:

''sort -f text.txt''

===== Обратная сортировка =====

''sort -r text.txt''

===== Сортировка по числовым значениям =====

Если строки содержат числа и их нужно сортировать как числа:

''sort -n text.txt''

===== Удаление повторяющихся строк =====

''sort -u text.txt''

===== Комбинирование =====

Для сортировки файла с игнорированием регистра, удалением дубликатов и сохранением результата:

''sort -fu text.txt -o text.txt''

===== Если файл ОЧЕНЬ большой =====

Надо добавить опцию ''--parallel=N'', где N — количество потоков для ускорения.

''sort -f text.txt --parallel=4''
