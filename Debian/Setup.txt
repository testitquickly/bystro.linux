===== Установить базовое ПО =====

Это можно поставить и c установочного диска Debian. 

==== Debian 12 ====

Из-под root:

{{{code: lang="sh" linenumbers="True"
apt install aptitude audacity curl fbreader gdu kdiff3 keepassxc kget krename krusader make mc most mpv network-manager-openvpn nfs-common onboard openconnect openssh-client openssh-server openvpn qalculate-gtk qbittorrent qmmp software-properties-common tellico trash-cli vlc vsftpd wget xsensors zim 
}}}

После установки ''software-properties-common'' можно расширить доступный набор пакетов в sources.list (тоже только из-под root):

{{{code: lang="sh" linenumbers="True"
apt-add-repository contrib non-free -y && apt update && apt upgrade && apt install rar unrar ethtool gir1.2-gtksource-4 iputils-ping iw smartmontools nvme-cli xdot wakeonlan -y && apt update && apt upgrade
}}}

==== Debian 13 ====

В Debian 13 отключена возможность ставить пакет ''software-properties-common'', потому что он тянет за собой Python-зависимости и сторонние PPA-механизмы, которые могут нарушить стабильность системы. Вместо этого Debian рекомендует использовать apt напрямую (/etc/apt/sources.list и /etc/apt/sources.list.d/) для добавления репозиториев, без PPA (типично для Ubuntu) и скриптов вроде ‘add-apt-repository‘.

Из-под root:

{{{code: lang="sh" linenumbers="True"
apt install aptitude 0ad audacity bzip2 curl easyeffects ethtool evince fbreader htop gdu gir1.2-gtksource-4 gzip iputils-ping iw k3b katomic kdiff3 keepassxc kget kmymoney krename krita krusader make mc most mpv mypaintncompress libasound2-plugins network-manager-openvpn nfs-common nvme-cli onboard openconnect openssh-client openssh-server openvpn p7zip-full qalculate-gtk qbittorrent qmmp rar smartmontools tar tellico timeshifttrash-cli unaceunrar unzip vlc vsftpd wakeonlan wget xdot xsensors xz-utils zim -y && sudo apt upgrade -y
}}}
