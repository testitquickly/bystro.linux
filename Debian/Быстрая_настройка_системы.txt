[ @debian @quicksetup ]

===== Установить основное ПО =====

''su''

C установочного диска Debian одной командой:

''apt install aptitude mc zim xsensors curl krusader keepassxc onboard openssh-server openssh-client openvpn openconnect network-manager-openvpn vsftpd audacity calibre qbittorrent krename jlha-utils arj unace rpm qmmp vlc smplayer kget kdiff3 locate qalculate-gt make wget''
