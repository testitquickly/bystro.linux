====== Настроить систему ======

[ @debian @quicksetup ]

===== Установить основное ПО =====

Из-под root:

1
это можно поставить и c установочного диска Debian:

''apt install aptitude audacity calibre curl gdu kdiff3 keepassxc kget krename krusader macchanger make mc mpv network-manager-openvpn onboard openconnect openssh-client openssh-server openvpn qalculate-gtk qbittorrent qmmp tellico software-properties-common vlc vsftpd wget xsensors zim'' 

2
после установки ''software-properties-common'' можно расширить доступный набор пакетов в sources.list:

'''
apt-add-repository contrib non-free -y && apt update && apt upgrade && apt install rar unrar iputils-ping smartmontools nvme-cli -y && apt update && apt upgrade
'''
