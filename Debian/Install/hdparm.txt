====== hdparm ======

[ @debian @install @hdparm ]

Программа для настройки параметров жёсткого диска для повышения производительности. 

Отображает параметры приводов SATA/IDE в Linux. Многие новые (начиная с 2008 года) USB-устройства поддерживают интерфейс SAT (SCSI-ATA Command Translation) и потому могут также настраиваться утилитой hdparm. 

''sudo aptitude install hdparm''
