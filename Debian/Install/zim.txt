====== zim ======

[ @debian @install @zim ]

Установить сразу с несколькими утилитами, которые будут нужны модулям:

''sudo apt install zim gir1.2-gtksource-4 xdot''

Можно поставить и через [[Software:flatpak]] или [[Software:snap]], но в нативной установке проще настроить внешнее оформление Zim (я подсел на тёмную тему).

Запустить, чтобы конфигурационные файлы создались. 

Закрыть.

Детальные настройки см. в [[Software:Zim]]
