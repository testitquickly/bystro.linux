====== PipeWire ======

[ @debian @install @sound @pipewire ]

Надо будет перезагрузить ноут.

Установить PipeWire и заодно включить ему BlueTooth.

* pipewire-pulse — замена PulseAudio. Его установка удалит и pulseaudio, и pulseaudio-bluetooth, поэтому пакеты для BlueTooth надо будет установить отдельно.
* pipewire-alsa — не нужен, если в системе будет pipeware-pulse, но может быть нужным старым приложениям, которые управляют звуком для PipeWire через ALSA API. Надо будет удалить pulseaudio-alsa.
* pipewire-jack — для роутинга звука через JACK.

''sudo apt install pipewire pipewire-audio pipewire-pulse pipewire-jack wireplumber libspa-0.2-bluetooth bluez bluez-tools bluedevil''

''systemctl --user --now enable pipewire pipewire-pulse wireplumber''

''systemctl --user --now disable pulseaudio pulseaudio.socket''

''sudo reboot''

Проверка

''ps aux | grep -E "pipewire|pulseaudio"''

Пример ответа:
