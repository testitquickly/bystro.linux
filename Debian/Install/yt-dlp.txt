====== yt-dlp ======

[ @debian @install @youtube ]

По-умолчанию в Debian установится древний ''yt-dlp'', а он уже немощный. 

===== Убрать ветерана =====

Если он есть, можно проверить версию:

''yt-dlp --version''

Ответ: 

''2025.04.30''

Убрать:

''sudo apt remove yt-dlp''

===== Поставить новую версию =====

''sudo curl -L https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp -o /usr/local/bin/yt-dlp && sudo chmod a+rx /usr/local/bin/yt-dlp''

Проверить:

''/usr/local/bin/yt-dlp --version''

Ответ «2026.02.04» радует.

Отныне и впредь запускать через 

''/usr/local/bin/yt-dlp''

Можно [[HowTo:Сделать:Симлинки и хардлинки:symlink]] на него у себя в ''~/.local/bin''

==== Добавить node.js ====

YouTube периодически изменяет механизм извлечения данных, поэтому может появиться WARNING «No supported JavaScript runtime could be found». Это ограничивает варианты для скачивания. 

Решение — установить фреймворк NodeJS (полностью; и внимание к номеру версии в установке, она уже может быть устаревшей): 

''curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - && sudo apt install -y nodejs''

Проверить:

''which node''

Ожидаем ''/usr/bin/node''

''which nodejs''

Ожидаем ''/usr/bin/nodejs''

В файл с конфигурацией для скачивания файлов через yt-dlp добавить параметр обращения к NodeJS: 

''--js-runtimes node:/usr/bin/node''

==== Проблема совместимости ====

Вполне вероятно, что yt-dlp и установленная версия NodeJS не будут совместимы, потому что софт постоянно развивается.

Проверить версию NodeJS:

''node --version''

Ставили 20*, поэтому в ответ ожидаемое: ''v20.20.2''

Запустить условное скачивание видео, в режиме симуляции:

''/usr/local/bin/yt-dlp --js-runtimes node --verbose --simulate https://www.youtube.com/watch?v=Sb4jBS-UyWE'' 

Смотреть в ответе:

''[debug] JS runtimes: node-20.20.2 (unsupported)''

надо удалить установленную и поставить другую:

''sudo apt purge nodejs''

''curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -''

''sudo apt install -y nodejs''

''node --version''

Запустить условное скачивание видео, в режиме симуляции:

''/usr/local/bin/yt-dlp --js-runtimes node --verbose --simulate https://www.youtube.com/watch?v=Sb4jBS-UyWE'' 

и проверить строку “JS runtimes“:

''[debug] JS runtimes: node-22.23.1''

===== Обновить =====

''sudo /usr/local/bin/yt-dlp -U''
