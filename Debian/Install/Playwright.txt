====== Playwright ======

[ @debian @install @playwright ]

Сперва надо решить, как его устанавливать: глобально или в каждый проект.

Можно устанавливать в каждый проект отдельно. Если проектов будет много, это приведет к дефициту свободного места на дисках, но IRL вряд ли придётся работать больше чем на нескольких проектах, а на современных дисках места больше, чем нужно.

Можно установить Playwright глобально в систему, а в конфигах каждого проекта прописывать путь к файлам Playwright. Может показаться правильным установить его глобально и задать ему интерпретатор Python через PATH в файле ''~/.bashrc'', но это будет действовать для всего ПО в системе, которое использует Python, и где-то что-то может засбоить. 

В современных OS нельзя устанавливать софт для Python напрямую, поэтому используем ''pipx'' — он тоже ставит пакеты в изолированное окружение, но оно будет доступно всем в системе:

''pipx install playwright''

Playwright требует headless-браузеры для работы

* Chromium → Chrome/Edge движок (Blink)
* Firefox → Gecko движок
* WebKit → Safari движок
 
Можно поставить только нужные:

''playwright install chromium firefox''

Если не перечислять нужное, то будут установлены все три.

Всё будет сохранено в ''~/.cache/ms-playwright/*''

Проверка (by default будет открыт Chrome):

''playwright codegen https://ro.wikipedia.org/wiki/Republica_Moldova''

Запуск см. [[Software:Playwright]]
