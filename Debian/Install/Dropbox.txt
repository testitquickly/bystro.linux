====== Dropbox ======

[ @debian @install @dropbox ]

https://www.dropbox.com/install — скачивание архива с установщиком.

Ещё когда-то предлагалось скачать deb-файл с клиентом Dropbox, но через консоль проще.

===== Установить =====

''sudo apt install libayatana-appindicator3-1 python3-gpg''

==== Служебный демон Dropbox ====

''cd ~ && wget -O - "https://www.dropbox.com/download?plat=lnx.x86_64" | tar xzf -''

==== GUI Dropbox ====

''cd ~ && wget -O ~/.dropbox-dist/dropbox.py "https://www.dropbox.com/download?dl=packages/dropbox.py" && chmod +x ~/.dropbox-dist/dropbox.py''

=== Остановить демон ===

Если вдруг был запущен:

''pkill dropbox''

=== Запустить Dropbox ===

Сперва демон, затем GUI

'''
~/.dropbox-dist/dropboxd && sleep 5; ~/.dropbox-dist/dropbox.py start
'''

и настроить подключение к аккаунту Dropbox. Например, сделать вход, настроить количество [[https://www.dropbox.com/account/security?_tk=account_menu|подключенных устройств]].

==== Проверить статус Dropbox ====

''~/.dropbox-dist/dropbox.py status''

Если он запущен, в ответ вернется “//Обновлено//”. Если нет — “//Dropbox isn't running!//”.

См. [[Software:Dropbox]]
