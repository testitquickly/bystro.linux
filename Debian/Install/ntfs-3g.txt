====== ntfs-3g ======

[ @debian @install @ntfs @windows ]

Утилита для доступа к файловой системе NTFS (стандарт для Windows). Как правило, поставляется вместе с Debian.

''sudo aptitude install ntfs-3g''

См. [[Software:ntfs-3g]]
