====== Whisper ======

[ @debian @install @machine_learning @openai @whisper ]

===== Установить Python =====

Нужен virtual environment для Python. Проверить, если он установлен: 

''apt list --installed python3-venv''

Если нет:

''sudo apt install python3-venv''

===== Установить ffmpeg =====

''sudo apt update; sudo apt install ffmpeg''

Этого достаточно. Если нужны какие-то особенные настройки и возможности, посмотреть на расширенную установку [[FFmpeg]]

===== Установить Whisper =====

==== Сделать каталог для Whisper ====

''mkdir ~/workspace/LLMs/Whisper/ && cd ~/workspace/LLMs/Whisper/''

==== Сделать virtual environment для проекта ====

''python3 -m venv whisper_env''

Активировать его

''source whisper_env/bin/activate''

//Неочевидное//: каталог  ''whisper_env'' нельзя __перемещать__ и переименовывать, по новому адресу вирутальное окружение не поднимется. Если очень надо его переместить, то:
 
* удалить существующий
* создать новый „virtual environment” с новым названием в том же каталоге

==== Снять проект из git ====

* Должен быть установлен консольный [[Software:git]]
* Мы всё ещё находимся в активированном окружении „''whisper_env''”. 

''pip install git+https://github.com/openai/whisper.git''

Если нет сообщений об ошибках, то всё готово.

Когда будет нужен выход из виртуального окружения Python:

1. зайти в каталог с этим окружением
2. скомандовать ''deactivate''

===== Обновить Whisper =====

Это будет редко, но иногда надо.

Узнать, если есть обновления — https://github.com/openai/whisper/releases/latest

Whisper установлен как пакет Python, поэтому надо а) поднять виртуальное окружение и уже там б) узнать установленную версию

''pip show openai-whisper | grep 'Version'''

и если решено обновлять, то:

''pip install --upgrade openai-whisper''

Также можно сделать

''pip install --upgrade --no-deps --force-reinstall git+https://github.com/openai/whisper.git''

См. [[Software:LLM:Whisper]]
