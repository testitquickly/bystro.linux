====== Slack ======

[ @manjaro @install @slack @yay ]

Можно установить из Pamac (gui-центр установки программ), но в будущем неизбежно возникнут сложности с обновлением. Slack будет настойчив в требованиях обновиться, но не факт, что в этот момент новейшая версия уже будет в общем репозитории. Поэтому лучше устанавливать из репозитория на Github через yay.

В репозиториях есть ещё утилита “slack”, которая вообще к слаку не относится. Нужен пакет “slack-desktop”. 

Установка через [[HowTo:Manage:Packages:yay]]

===== Обновить систему =====

Дежурный шаг, но не помешает:

''sudo pacman -Syu''

===== Установить =====

''yay -S slack-desktop''

Если хочется «тихой установки», нужен параметр --noconfirm:

''yay -S slack-desktop --noconfirm''

Если всё в норме, можно проверить версию slack:

''slack -v''

===== Запустить =====

Из терминала: 

''slack &''

Лучше запускать всё из общего меню приложений

Логин будет происходить через браузер.

===== Обновить =====

''yay -Syu --devel --timeupdate''

===== Удалить =====

''yay -Yc''

Можно удалить его и в режиме «тишины»

''yay -Rns slack-desktop --noconfirm''

но лучше удалять все глядя в монитор, чтобы не осталось ненужных зависимостей.
