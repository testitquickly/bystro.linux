====== Узнать свой внутренний IP ======

[ @manjaro @console @ip ]

===== Internal IP =====

''ifconfig $(route | awk '{if($1=="default") print $NF}') | awk '{if($1=="inet") print $2}'''

===== External IP =====

''curl ifconfig.co''

См. отдельно для Debian — [[:Debian:HowTo:Проверить:Свой IP]]
