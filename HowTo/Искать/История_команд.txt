====== История команд ======

[ @console @search ]

Во-первых, надо проверить, сколько команд в консоли сохраняется.

''Konsole''
''> Редактирование профиля пользователя''
''> Журнал''
''> Сохранять последние «300 строк»''

===== Посмотреть весь список =====

''history | less''

Постраничное пролистывание нужно, потому что список наверняка будет излишне большим.

Выход по клавише ''q''.

==== Выполнить команду из сохраненных ====

Например, команду номер 23

''!23''

==== Удалить одну из команд ====

''history -d 23''

==== Удалить весь список ====

''history -c''

===== Поиск по истории =====

Нажать Ctrl+r — появится поле для поиска с названием 

''(reverse-i-search)`':'' 

Начать ввод текста — автоматически появятся подходящие строки из истории..

Нажатие на ''Tab'' выведет выбранную команду, останется нажать ''Enter''.
