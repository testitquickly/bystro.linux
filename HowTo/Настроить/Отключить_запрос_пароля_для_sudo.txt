====== Отключить запрос пароля для sudo ======

[ @console @sudo @password ]

Можно (хотя и не нужно) вообще отключить запрос пароля при выполнении команд через sudo:

''sudo mcedit /etc/sudoers.d/nopasswd''

Вставить:

__''имя_пользователя''__ ''ALL=(ALL) NOPASSWD: ALL''

Сохранить и выйти.

Выставить этому файлу права:

''sudo chmod 440 /etc/sudoers.d/nopasswd''

Проверить синтаксис файла

''sudo visudo -c''

Ждём в ответ 

''/etc/sudoers.d/nopasswd: parsed OK''

иначе алярм.

Закрыть окно Konsole или просто открыть новое — настройки подхватятся моментально. 

Если уже пошли по столь радикальному пути, можно даже удалить файл с таймаутом:

''sudo rm /etc/sudoers.d/timeout''
