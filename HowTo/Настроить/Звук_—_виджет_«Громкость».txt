====== Звук — виджет «Громкость» ======

[ @sound ]

Правой кнопкой мыши по виджету «Громкость» в трее 
''> Настроить виджет «Громкость»''
''> Основное'' 
''> Шаг изменения громкости = 5%'' 

Убрать галочку «Воспроизводить звук при регулировке устройств»

'''
Звуковые устройства 
> Приложения 
'''

Замьютить «Звуки уведомлений»

'''
Звуковые устройства 
> Дополнительно
'''

Убрать галочку для «Добавить виртуальное устройство для параллельного вывода на все звуковые карты»

Убрать галочку для «Автоматически переключать все потоки при появлении нового устройства»
