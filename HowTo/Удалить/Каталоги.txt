====== Каталоги ======

[ @console @folders ]

Для удаления каталогов используется команда ''rmdir'', но она разрешает удалять можно только пустые каталоги, т. е. не содержащие файлов и поддиректорий (вложенных папок), поэтому используется редко.

Чтобы просто удалить каталог можно использовать общую команду ''rm'' с ключом ''-r'' (recursive). 

''rm -r /mnt/save/alldocuments/''

Внимание, ''rm'' удаляет файлы __сразу__, без возможности восстановления.
