====== Исправить битые пакеты (held) ======

[ @console @upgrade @grub ]

Если после обновлений (или apt autoremove) какие-то пакеты были удалены, и при установке нового софта система сообщает "Невозможно исправить ошибки, у вас отложены (held) битые пакеты":

'''
sudo atp-get autoremove
sudo apt autoclean
sudo dpkg-reconfigure -a
sudo apt upgrade
'''

Внимание, будет много вопросов для настройки (по-сути, система обновляется как будто с нуля). Потом будет подтянут ряд пакетов из репозиториев.

Осторожно, может быть установлен старый grub.
